COVID-19 Advisory System

Overview
The COVID-19 Advisory System is a web-based application that analyzes user symptoms and provides personalized health guidance, risk assessment, and precautionary measures.

Disclaimer: This tool is for informational purposes only and is not a substitute for professional medical advice.

---

Features

- Symptom tracking with dynamic selection
- Patient information input (age, duration, vaccination status)
- Risk score calculation with visual dashboard
- Severity classification and recommendations
- Medical plan and precautionary guidance
- Vaccination and hygiene recommendations
- Dietary suggestions
- Emergency alerts for critical conditions
- Long COVID risk analysis
- Regional risk tracker (mock data)
- Nearby clinics display (mock data)
- Mental health support module
- Export/Print report functionality

---

Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
